# 🌟 Ashravi Web

**Modern parenting education platform built with Next.js 14, TypeScript, Redux Toolkit, and Tailwind CSS.**

Ashravi Web empowers parents with evidence-based courses, personalized learning paths, and AI-powered assessments for child development.

## ✨ **Features**

### **User Features**
- 🔐 **Authentication**: Email/phone login, social OAuth (Google, Facebook, Apple)
- 📚 **Course Catalog**: Browse 500+ parenting courses with advanced filters
- 🎯 **Personalized Learning**: AI-powered course recommendations
- 👶 **Child Profiles**: Track multiple children's development milestones
- 📊 **Progress Tracking**: Monitor course completion and achievements
- ❤️ **Wishlist**: Save courses for later
- 🌙 **Dark Mode**: System-aware theme switching
- 📱 **Responsive Design**: Mobile-first, tablet, and desktop optimized

### **Technical Features**
- ⚡ **Next.js 14 App Router**: Server & client components, ISR, SSR
- 🔄 **Redux Toolkit**: Centralized state management with thunks
- 🎨 **Design Tokens**: Consistent theming with CSS custom properties
- 🧪 **Testing**: Vitest (unit) + Playwright (E2E)
- 📈 **Performance**: Optimized images, code splitting, lazy loading
- ♿ **Accessibility**: WCAG 2.1 AA compliant
- 🔍 **SEO**: JSON-LD structured data, meta tags, sitemap

## 🛠️ **Tech Stack**

### **Core**
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router, TypeScript)
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + Custom CSS Modules
- **Data Fetching**: [SWR](https://swr.vercel.app/) (client-side caching)

### **Development**
- **Language**: TypeScript (strict mode)
- **Linting**: ESLint + Prettier
- **Git Hooks**: Husky + lint-staged
- **Testing**: Vitest + React Testing Library + Playwright

### **Infrastructure**
- **Package Manager**: pnpm
- **Node Version**: 18.x or higher
- **Deployment**: Vercel (recommended)
